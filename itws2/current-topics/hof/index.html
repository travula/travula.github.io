4 Functions as return values

Here it might help to follow the signatures of some common higher order functions.

4.1 compose

A, B, C are arbitrary types. 

f : B -> C
g : A -> B
(compose f g) : A -> C
  • Example of using compose 
((compose add1 mul2) 4) ; => 9
((compose mul2 add1) 4) ; => 10

4.2 derivative

Signature 

derivative : [real? -> real?] ->
             [real? -> real?]
  • Example of using derivative 
((derivative add1) 2)
((derivative mul2) 5)

4.3 integral

integral : [[real? -> real?], 
             real? real?] -> 
           real?

This signature of integral emphasizes that integral of a function is a function that when given two numbers (lower and upper bounds) returns a number. 

integral : [[real? -> real?]] -> 
             [[real? real?] -> 
              real?]

6 derivative

(define derivative
  (let ([dx 0.1])
    (lambda (f)   ; f: real? -> real?
      (lambda (x) ; x: real?
        (/ (- (f (+ x dx))
              (f x))
           dx)))))

7 Patterns to function parameters

;;; map-add1 takes a list and adds 1 to 
;;; each element

(define map-add1
  (lambda (ls)
    (cond [(null? ls) '()]
          [else 
           (cons (add1 (first ls))
                 (map-add1 (rest ls)))])))

(define map-mul2
  (lambda (ls)
    (cond [(null? ls) '()]
          [else 
           (cons (mul2 (first ls))
                 (map-mul2 (rest ls)))])))

7.1 Pattern

Notice the pattern:

  • add1 vs mul2
  • map-add1 vs. map-add2

7.2 Abstract structure

(define map-<f>
  (lambda (ls)
    (cond 
        [(null? ls) '()]
    [else 
     (cons (<f> (first ls))
           (map-<f> (rest ls)))])))

7.3 Map

;;; list-map : [A->B (listof A)] ->
;;;            (listof B)

(define list-map
  (lambda (f ls)
    (cond 
      [(null? ls) '()]
      [else (cons 
             (f (first ls))
             (list-map f (rest ls)))])))

(list-map add1 '()) ; => '()
(list-map add1 '(2 4 8)) ; => '(3 5 9)

8 List map generator

Takes a function f and returns a function that lifts f to operate on lists. 

;;; list-map-generator: [A->B] ->
;;;                     [(listof A) -> 
;;;                       (listof B)]
(define list-map-generator
  (lambda (f)
    (lambda (ls)
      (cond 
        [(null? ls) '()]
        [else (cons 
               (f (first ls))
               ((list-map-generator f)
                (rest ls)))]))))


(define list-map-add1 
  (list-map-generator add1))

9 Example of using local recursion

Here is a version of list-map-generator which uses a helper function. The helper function is local to list-map-generator. Notice that the helper only takes ls as the parameter; there isn't a need to parameterize helper on f because f is constant as far as helper is concerned. 

;; Usage
(define map-add1 (list-map-generator/helper add1))

(define list-map-generator/helper
  (lambda (f)
    (letrec ([helper
              (lambda (ls) 
                (cond  
                  [(null? ls) '()]
                  [else (cons
                         (f (first ls))
                         (helper (rest ls)))]))])
      helper)))
