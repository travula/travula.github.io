1 Environments specify bindings

  • An environment maps identifiers to denotations (not expressions). Often these denotations are values.
  • The identifier-value pair is called a binding (contrast with bind). 
{x: 3, y: '(a b c), z: #t}

2 Environment composition

  • r1.r2 is the composition of r1 with r2. 
Let r1 = {x: 5, y: 3},
    r2 = {x: 7, z: 8}

r1.r2 = {x: 5, y: 3, z: 8}

Composition is not commutative. r1.r2 is not equal to r2.r1.

2.1 Where are environments used?

Environments are used as input to evaluate an expression. An expression is evaluated with respect to an environment.

The environments supplies the values of the free variables in the expression being evaluated.

Looking up an environment to find the value of an identifier is equivalent to substituting the identifier with its binding value. 

(+ x 2)  wrt {x:4} =  ; substitute
(+ 4 2) =             ; simplify
6                     ; answer
