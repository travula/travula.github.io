1 Objective of Lab

  • Learn the basic mechanics of working with the Racket Programming Language environment. Creating and saving .rkt files. Adding module declarations. Adding provide declarations.
  • Apply the methodology of program design (Purpose, Usage, Signature, Code).
  • Apply the idea of structural recursion pattern on natural numbers and lists.
  • Understand difference between locally defined recursive functions (using letrec) and top-level define recursive function (using define).

2 Problems

Write the purpose, usage, contract and code for the following problems:

Caution!
Please use the exact names for your functions as mentioned below in each question. Your solutions will not be graded if different names are used.

2.1 ring-area

Write a function called ring-area that takes two arguments (outer-radius and inner-radius) and returns the area of the ring.

2.2 list-with-length-at-least-two?

Define list-with-length-at-least-two? that returns true if the list has at least two elements, false otherwise.

2.3 remove-second

Write a function remove-second that takes a list of at least two elements and returns a list without the second element of the original list. (Hint: use the predicate above to define the contract for remove-second.)

2.4 list-length

Write a function list-length that computes the length of a list using structural recursion on lists.

2.5 gcd

Write a function gcd that takes two natural numbers and returns their greatest common divisor.

2.6 gcd/letrec

Rewrite the function gcd/letrec that also computes gcd but has a locally defined function using letrec.

3 Instructions

Save the above problem solutions in a single file called main.rkt.
